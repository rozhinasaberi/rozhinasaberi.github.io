# rozhinasaberi.github.io

A static personal web repository containing resume materials, academic documents, and supporting assets for a GitHub Pages-style site.

## What This Project Is About

This repository appears to be an older personal or academic website container rather than a full software product. It includes resume-related documents, images, and a static HTML/CSS page inside the `cv/` folder.

This is a **static personal website / document hosting repository**.

## Contents

- `cv/midterm.html` - main HTML page
- `cv/midterm1.css` - styling
- resume and academic PDF files
- supporting image assets

## Tech Stack

- HTML
- CSS

## Open Locally

You can open the page directly in a browser or serve the folder with a simple local server.

Example:

```bash
cd cv
python3 -m http.server 5500
```

Then open `http://localhost:5500`.
